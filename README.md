# Bitcoin Đà Nẵng

Static website for the Bitcoin Đà Nẵng meetup community — the first Bitcoin-only community in Đà Nẵng, Việt Nam.

- **Primary domain:** [bitcoindanang.com](https://bitcoindanang.com)
- **Email domain:** bitcoindanang.org
- **Contact:** contact@bitcoindanang.org

## Stack

- Plain HTML / CSS / JS (no framework) — mobile-first, bilingual (VI primary, EN secondary)
- [TinaCMS](https://tina.io) for the blog (Phase 6)
- [Cloudflare Pages](https://pages.cloudflare.com) for hosting (Phase 7)

## Pages

- `/` — meetup homepage (dark theme, gold-on-black)
- `/pizza-day-2026.html` — Bitcoin Pizza Day landing (light theme, live price)
- `/bitcoin.html` — "Bitcoin Là Gì?" educational page (light theme)

## Brand

- Gold `#D4AF37`, near-black `#0D0D0D`, warm white `#FAF7F0`
- Typography: Cormorant Garamond (display) + DM Sans (body)
- Logo: gold Bitcoin coin with 33 stars (33 Vietnamese provinces) on the outer ring
